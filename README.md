# NixOS config by jakublaba

## Installation
> [!WARNING]
> Before building, replace `hardware-config.nix` in chosen host with the one that was generated by you in `/etc/nixos/hardware-config.nix`. \
> Skipping this step will most likely result in bricking your system ðŸ§±

### Get a fresh hardware scan.
```
sudo nixos-generate-config
```
You will get a warning like this
```
warning: not overwriting existing /etc/nixos/configuration.nix
```
But we only care about `hardware-configuration.nix` here so it's safe to ignore it.

### Overwrite `hardware-config.nix` from this repo with your own
```
sudo cp /etc/nixos/hardware-config.nix ./hosts/[pc|vm]/core/hardware-configuration.nix
```

### Build
```
sudo nixos-rebuild switch --flake .#[pc|vm]
```
